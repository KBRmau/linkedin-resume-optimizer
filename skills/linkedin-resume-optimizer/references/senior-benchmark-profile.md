# Perfil de referência — densidade de dado sênior

Este é um perfil real (Data Engineer sênior) fornecido pelo usuário como
exemplo de perfil que já recebe bastante atenção de recrutadores. Use como
régua: ao otimizar qualquer outro perfil, mire nesta densidade de dado
mensurável por bullet — não copie o conteúdo, copie o **padrão**.

O que este perfil faz certo, bullet a bullet:

- Cada bullet de experiência tem pelo menos um número: escala (`2,000+
  pipelines`, `500+ Airflow/Composer DAGs`), volume (`~66K task events per
  day`), taxa de sucesso (`~70% of recurring low-risk KTLO incidents`),
  contagem de recursos provisionados (`1,000+ GCP data resources`), ou tempo
  (`from 2 days to 10 minutes`).
- Máximo de 5-6 bullets por experiência — nenhum cargo tem uma lista infinita
  de responsabilidades genéricas.
- O About segue os 4 blocos: gancho (cargo + anos + foco) → escala do papel
  atual (com números) → histórico anterior relevante (com resultado
  mensurável por empresa) → linha final densa em stack/palavra-chave.
- O headline é `função | especialização | stack`, sem frase de efeito.
